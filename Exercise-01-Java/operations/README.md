# Linux File Operations

## Aim
To perform basic Linux file operations such as renaming files, listing files, and displaying file contents.

## Commands Used

### Rename file

```bash
mv students.csv kavin.txt
```

### Display file details

```bash
ls -l
```

### Display file content

```bash
cat kavin.txt
```

## Explanation

- `mv` → Renames or moves files
- `ls -l` → Shows detailed file information
- `cat` → Displays file content
- `type` → Identifies shell commands/programs
